🥇 СПОСОБ 1: GITHUB (РЕКОМЕНДУЕМЫЙ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Создать публичный репозиторий на GitHub:

1️⃣ Зайдите на github.com
2️⃣ Создайте аккаунт (если нет)  
3️⃣ Нажмите "New repository"
4️⃣ Назовите: "roulette-data-collector"
5️⃣ Поставьте галочку "Public"
6️⃣ Нажмите "Create repository"

В PowerShell выполните:

cd C:\Users\Pablo\RULETTT
git init
git add .
git commit -m "Автоматический сборщик данных рулетки"
git branch -M main
git remote add origin https://github.com/ВАШЕ_ИМЯ/roulette-data-collector.git
git push -u origin main

✅ Результат: Любой человек сможет скачать весь проект!

Создать архив для отправки:

1️⃣ Правый клик на папке C:\Users\Pablo\RULETTT
2️⃣ "Отправить" → "Сжатая ZIP папка"
3️⃣ Переименуйте в "Roulette_Collector.zip"
4️⃣ Отправьте через:
    - Email (если файл < 25MB)

    - Telegram
