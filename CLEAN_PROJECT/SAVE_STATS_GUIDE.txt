📈 showRouletteStats()
   └─ Показать статистику в консоли (не сохраняет)

💾 exportRouletteStats()
   └─ Экспорт статистики в текстовом формате
   └─ Копирует в буфер обмена
   └─ Готово для вставки в файл

📋 exportRouletteData()
   └─ Экспорт данных в JSON формате
   └─ Для Python анализа (py console_to_analysis.py)
