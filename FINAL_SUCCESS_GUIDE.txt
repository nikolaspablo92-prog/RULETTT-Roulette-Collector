📋 ТЕСТ 1: ПРОВЕРЬТЕ СРЕДУ РАЗРАБОТКИ

В вашем Codespace выполните:

1️⃣ ОТКРОЙТЕ ТЕРМИНАЛ:
   • В Codespace: Terminal → New Terminal

2️⃣ ПРОВЕРЬТЕ PYTHON:
   python --version
   # Должен показать Python 3.11+

3️⃣ ПРОВЕРЬТЕ УСТАНОВЛЕННЫЕ ПАКЕТЫ:
   pip list
   # Должны быть установлены зависимости из requirements.txt

4️⃣ ЗАПУСТИТЕ ВЕБ-СЕРВЕР:
   python -m http.server 8000
   # Codespace автоматически откроет порт

5️⃣ ОТКРОЙТЕ ВЕБ-ИНТЕРФЕЙС:
   • Нажмите на уведомление "Open in Browser"
   • Или перейдите во вкладку "Ports" и откройте порт 8000

📋 ТЕСТ 2: ПРОВЕРЬТЕ КОЛЛЕКТОР РУЛЕТКИ

1️⃣ ОТКРОЙТЕ ФАЙЛ:
   auto_collector_console_code.js

2️⃣ СКОПИРУЙТЕ ВЕСЬ КОД:
   Ctrl+A → Ctrl+C

3️⃣ ОТКРОЙТЕ САЙТ КАЗИНО:
   • Stake.com, BC.Game или любую европейскую рулетку
   • В новой вкладке браузера

4️⃣ ВСТАВЬТЕ КОД В КОНСОЛЬ:
   • F12 → Console
   • Ctrl+V → Enter
   • Должно появиться: "🎲 Коллектор рулетки запущен!"

5️⃣ ПРОВЕРЬТЕ КОМАНДЫ:
   showRouletteStats()    // Покажет статистику
   saveRouletteStats()    // Скопирует в буфер
